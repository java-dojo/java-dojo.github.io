Introduction

This is a Java code dojo.

You will find code katas of different levels.

The ideas for these katas are mostly based on programming tasks from Rosetta Code.

Basically it should be possible to solve such a kata in about 30 to 60 minutes.

The following levels are available (at the moment):

  • Beginners

Development Requirements

To perform these katas you need a JDK 8 and Maven 3.3.x.
